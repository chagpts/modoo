US TAX SERVICE 페이지 파일 안내

이 ZIP에는 index.html과 서비스별 HTML 페이지가 포함되어 있습니다.

Hero 배경 영상은 YouTube가 아니라 자체 mp4 파일을 사용합니다.
광고 없이 사용하려면 assets 폴더에 아래 두 파일을 넣어주세요.

1. assets/nyc-skyline1.mp4
2. assets/nyc-skyline2.mp4

HTML에서는 캐시 방지를 위해 아래 경로를 사용합니다.
./assets/nyc-skyline1.mp4?v=20260620-02
./assets/nyc-skyline2.mp4?v=20260620-02

두 영상은 자동으로 교차 재생됩니다.
서버 업로드 시 HTML 파일과 assets 폴더를 함께 덮어써주세요.
