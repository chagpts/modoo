8단계 AI 전산감사 실습 로드맵

회계 데이터가 생성되고, 통제되고, 시스템에서 처리되고, 감사되고, 보안까지 검증되는 흐름으로 구성했습니다.

01

회계 데이터 이해

거래, 증빙, 분개, 전표, 총계정원장, 재무제표의 연결 구조를 실습합니다.

  • 영수증 기반 비용 분개
  • 세금계산서 매입/매출 처리
  • 계정과목 오류 찾기
02

내부회계관리

승인, 권한, 증빙, 변경 이력 등 회계 오류를 막는 핵심 통제를 점검합니다.

  • 전표 승인 분리 점검
  • 증빙 누락 탐지
  • 결산 체크리스트 검토
03

AI OCR 회계 자동화

AI가 증빙을 읽고 금액, 거래처, 세액, 계정과목을 추출해 자동 분개하는 과정을 구현합니다.

  • OCR 결과 검증
  • 계정과목 추천
  • 자동 분개 승인 흐름
04

ITGC & ITAC

시스템 권한, 변경관리, 운영 접근, 자동통제의 설계와 작동 여부를 확인합니다.

  • 퇴사자 계정 탐지
  • 운영 DB 접근 점검
  • 한도 초과 자동통제 테스트
05

전산감사 데이터 분석

회계 데이터와 로그를 분석해 오류, 부정, 이상거래, 통제 미비를 탐지합니다.

  • 중복 지급 분석
  • 휴일·야간 전표 탐지
  • 특정 거래처 집중 분석
06

정보보안 회계 시스템 감사

ERP, DB, 계정, 접속 로그, 데이터 반출 이력을 보안 감사 관점으로 점검합니다.

  • 관리자 권한 검토
  • 비정상 로그인 탐지
  • 회계 데이터 다운로드 분석
07

종합 케이스 감사

가상의 기업 데이터를 바탕으로 감사 조서와 개선 권고, 최종 보고서를 작성합니다.

  • 문제 상황 분석
  • 감사 증거 정리
  • 감사보고서 제출
08

커리어 & 포트폴리오

실습 결과물을 전산감사, 내부감사, IT감사, 회계 자동화 직무 포트폴리오로 연결합니다.

  • 프로젝트 산출물 정리
  • 이력서 문장 작성
  • 면접 질문 대비
Practice Lab
